# Asset Register

Status: working register.

## Rules

- Record source, license or ownership basis, usage location, and review status for public imagery and portal-facing media.
- Do not use manufacturer logos, aircraft tail numbers, customer marks, or identifiable private aircraft details unless AMG has the right to use them.
- Representative aircraft images must not imply endorsement, affiliation, ownership, operator authority, or availability.

## Current Sources

- Public AMG brand assets in `public/images`.
- Generated or edited media in `public/images/amg-generated`.
- Interface icons from project dependencies such as `lucide-react`.

## Review Need

Run a media review before production launch to confirm all public imagery has an ownership, license, generated-media, or fair-use rationale recorded.
