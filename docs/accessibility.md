# Accessibility Notes

Status: implementation notes.

The public layout includes a skip link to `#main-content`. Forms include visible labels, required indicators, accessible notices, semantic sections, and keyboard-operable native inputs. The cookie preferences modal uses `role="dialog"` and `aria-modal="true"`.

## Standards Target

AMG should review public and portal interfaces against WCAG 2.2 AA where practical, including keyboard access, visible focus, color contrast, form labels, error messages, headings, landmarks, and responsive text fit.

## Feedback

Accessibility feedback should be routed to `information@amgaviationgroup.com` with page URL, issue description, device/browser, and assistive technology when applicable.
