# Media Review Checklist

- Every public asset has an ownership, license, generated-media, or approved-use basis.
- AI-generated assets include source/tool, prompt or production brief, review status, and usage page.
- Assets with people, readable aircraft tails, manufacturer logos, airport identifiers, or client/vendor marks are reviewed before publication.
- Assets marked `remove_before_launch` are not referenced by public pages.
- Aircraft images are representative category references unless a written agreement says otherwise.
- Unknown-license assets are marked needs review and are not treated as approved.
