# AMG Aviation Group

Next.js/Vercel website for AMG Aviation Group.

## What is included

- Public website inspired by premium private aviation editorial sites
- Pages for Home, About, Services, Aircraft, Pilot Network, Plans, Team, and Contact
- Direct member login at `/login`
- Role-based portal entry pages:
  - `/portal/client`
  - `/portal/crew`
  - `/portal/admin`
- Preserved `cms/` reference data from the previous repository

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

The project is ready for Vercel deployment from the `main` branch.
